Mission of the project:
- To create an interactive website. 
- There will be a question to user, for example, "Will you marry me?", and there are green Yes button and red No button. 
- When the user click no, the Yes button will grow bigger and gradually leaning towards the center of the monitor while the No button will grow smaller. Continue this process multiple times. The number of times is instructed below "For the red no button". After the last time. The Yes button will be at the center of the page while the No button will start changing position. Then when the user's mouse cursor moving to the No button, the No button will jump to another location so the user wouldn't be able to click the No button, until the user click the Yes button, then both buttons disappear and a ending view shows texts:  "Knew you would say yes".
